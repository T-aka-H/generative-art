# Waves

**インタラクティブ・ジェネレーティブ・オーシャン**

p5.js で描かれた、リアルタイムに変化し続ける海のジェネレーティブアート。時刻、位置情報、デバイスのセンサー、そしてあなたの指先が、世界にひとつだけの海の風景を生み出します。

**Live Demo:** https://t-aka-h.github.io/generative-art/

---

## アート性について

### コンセプト: 「触れる海」

このアートは、**観る人が同時に参加者になる**というコンセプトで作られています。従来の絵画や映像は一方向的ですが、Waves は観る人の操作・環境・時間によって常に姿を変えます。同じ風景は二度と現れません。

### 時間と空間の融合

- **時刻連動**: 現実の時間がそのまま空と海の色に反映されます。朝焼けの紫とオレンジ、昼の白い光、夕暮れの赤橙、夜の深い紺。現実の時間の流れの中に、デジタルの海が存在します。
- **位置連動**: あなたがいる緯度に応じて海の色味が変わります。赤道に近ければ明るいターコイズの熱帯の海、高緯度なら暗い紺の北極の海。どこで開いても、その場所に適した海が現れます。
- **コンパス連動**: デバイスの向きに応じて波が流れる方向が変わります。東を向けば波は右へ、西を向けば左へ。物理的な身体の向きがデジタルの海に反映されます。

### 波の造形

120 層の波が重なり、各層は **Perlin ノイズ** と **sin 波** の合成で形成されています。Perlin ノイズは自然界のゆらぎを模倣する数学的手法で、完全なランダムではなく「連続的な不規則さ」を生みます。これにより、本物の海の波のような有機的な曲線が生まれます。

波頭には白い泡（foam）と飛沫（spray）がノイズベースで配置され、水面には光のきらめきが明滅します。これらはすべて数式から生成されるもので、画像素材は一切使用していません。

### 音の生成

環境音もコードで生成しています。Web Audio API でホワイトノイズを作り、ローパスフィルタを通して波の音に変換。すべてがブラウザ内でリアルタイムに合成されます。

## 操作方法

### デスクトップ

| 操作 | 効果 |
|------|------|
| マウス左右移動 | 波の流れる方向と速度 |
| マウス上下移動 | 波の穏やかさ / 荒さ |
| クリック（海面） | 波紋が広がる |
| クリック（空） | 時間帯を 2 時間進める |
| ダブルクリック（空） | 実時刻に戻す |

### スマートフォン

| 操作 | 効果 |
|------|------|
| タッチ | 波紋が広がる |
| スワイプ | 突風が波を押す |
| 長押し | 水面を押し込む（白いリング表示） |
| ピンチ（2本指を広げる） | 波が荒くなる |
| ピンチ（2本指をつまむ） | 波が穏やかになる |
| 端末を振る | 大きな波が発生 |
| 端末の傾き | 波の振幅と流れ |
| コンパス方向 | 波の流れる方向 |
| 空をタップ | 時間帯を 2 時間進める |
| 空をダブルタップ | 実時刻に戻す |

## セットアップ

### ブラウザで開く

https://t-aka-h.github.io/generative-art/ にアクセスするだけで動作します。

### iPhone でフルスクリーン体験

1. Safari で上記 URL を開く
2. 共有ボタン（箱から矢印のアイコン）をタップ
3. 「ホーム画面に追加」を選択
4. ホーム画面から「Waves」アイコンで起動

ホーム画面から起動すると、アドレスバーやタブのない全画面表示になります。

## 技術構成

| 技術 | 用途 |
|------|------|
| [p5.js](https://p5js.org/) | Canvas 描画 |
| Perlin Noise | 波の不規則な形状 |
| Catmull-Rom Spline | 滑らかな波の曲線 |
| Web Audio API | 環境音の生成 |
| DeviceOrientationEvent | 傾き・コンパス検知 |
| DeviceMotionEvent | シェイク検知 |
| Geolocation API | 緯度による海の色変化 |
| GitHub Pages | ホスティング |

### ファイル構成

```
index.html           - HTML / CSS / スタンドアロンモード制御
sketch.js            - p5.js メインスケッチ（描画・操作・音声）
apple-touch-icon.png - ホーム画面用アイコン (180x180)
icon.svg             - アイコンの SVG ソース
```
